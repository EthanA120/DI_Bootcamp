Lets demonstrate some properties we can use to style the page:

.ClassName will style the class that its name come after the dot

All the paragraphs in "RedPar" class will turn to red.

All the elements in this div section will be in the color green due to the ID style reference

And all the elements within the div section will be oblique due to the parent and child relationship

.class - Refer to class element
#id - Refer to id element
* - Refer to all elements due to cascading rules
element - Refer to reference
element, element - Refer to more than one element
element element - Refer to an element(2) inside an element(1)
element > element - Refer to an element(2) that is a direct child of an element(1)
element + element - Refer to an element(2) that comes right after an element(1)
:hover - Valid only when cursor is hovering over the reference
:first-child - Refer to first child
:last-child - Refer to last child
!important (not recommended) - Overrule the cascading
